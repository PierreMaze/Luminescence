# 🍸 Bar & Event Space - Plateforme Web

## 📝 Description du Projet

Plateforme web dynamique pour un établissement multifonctionnel combinant bar, espaces locatifs et services événementiels. Le lieu propose une expérience unique en réunissant sous un même toit des espaces modulables pour différents usages professionnels et festifs.

## 🌟 Caractéristiques Principales

### 🍹 Services de Bar

- Service de boissons et cocktails
- Carte des consommations

### 📍 Espaces à Louer

- Local de 150m² modulable
- Open-space pour événements professionnels
- Rooftop avec vue panoramique
- Cabinet de soin équipé
- Calendrier de disponibilité

### 🎉 Services Événementiels

- Organisation de masterclass
- Concerts et performances live
- Événements privés et professionnels
- Agenda des événements

## 💻 Fonctionnalités Techniques

- Interface responsive (mobile-first)
- Mise à disposition la carte
- Utilisation de google calendar pour afficher les événement à venir
- Formulaire de contact
- Newsletter

## 🛠️ Technologies Utilisées

- Frontend : React, Tailwind, react-router-dom, react-google-recaptcha, date-fns, prop-types, ...
- Backend : NodeJS, Express.
- Hébergement : Hostinger.

## 📥 Installation et Scripts

Après avoir cloné le projet, installez les dépendances :

```bash
npm install
```

Scripts disponibles :

```bash
npm run dev      # Lance l'application en mode développement (port 3000)
npm run build    # Compile le projet pour la production
npm run lint     # Vérifie le code avec prettier et eslint
npm run fix      # Corrige automatiquement le formatage et le lint
npm run clean    # Supprime node_modules, dist, et package-lock.json
npm run preview  # Prévisualise le build de production
```

Pour arrêter le serveur de développement, utilisez `Ctrl + C`.
